# Ungdomsprogrammet
Et fulltidsprogram for personer mellom 18 og 30 år som trenger hjelp fra Nav til å komme ut i jobb. Dette er et forsøk som testes på et utvalg Nav-kontorer i Agder, Vestland, og Vestfold og Telemark.

## DBT prosjektet
### Hvordan kjøre lokalt, e.g. Knast
dbt run --select UP_utpakking.*
